# Gestión de Finanzas Personales

Proyecto académico para el desarrollo de una aplicación web orientada al control de ingresos, gastos y presupuestos personales.

---

## Objetivo

Permitir a los usuarios registrar sus transacciones financieras y visualizar un resumen de su situación económica mediante reportes básicos.

---

## Estructura del Proyecto

```
finanzas-personales/
├── frontend/     → Interfaz de usuario de la aplicación
├── backend/      → Lógica del servidor y gestión de datos
└── database/     → Scripts y diseño de la base de datos
```

---

## Descripción General

La aplicación facilitará la administración de las finanzas personales, permitiendo al usuario llevar un control organizado de sus movimientos económicos y apoyar la toma de decisiones financieras.

---

## Funcionalidades previstas

- Registro de ingresos y gastos
- Categorización de transacciones
- Visualización de reportes y gráficos
- Gestión de presupuestos mensuales
